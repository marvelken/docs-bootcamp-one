# Documentation Bootcamp Portfolio

Welcome! This is a documentation portfolio built during a technical writing bootcamp. It showcases two key deliverables: an API reference and a hands-on tutorial.

## What's Inside

- **[GitHub API Reference](./api-reference)** — A complete reference for three GitHub REST API endpoints: Get a User, Get a Repository, and List Repository Issues.
- **[Make Your First API Call with Axios](./tutorial-axios)** — A step-by-step tutorial that walks you through installing Axios, making GET and POST requests, handling errors, and reading an annotated TypeScript code sample.
